Mozilla FirefoxOS
=======
App para a hackaton da Mozila na #CPBR7.

TwitMemory
=======
Digitando seu perfil do Twitter, o app gera um jogo da memória com seguidores aleatórios. Um temporizador é iniciado para marcar o tempo do jogo.
